# Example 1

Following http://simon-rohou.fr/research/tubex-lib/doc/install/03-start-cpp-project.html

To execute:
First, run [VIBes-viewer](https://github.com/ENSTABretagneRobotics/VIBES):

```bash
VIBes-viewer
```

Then build and run

```bash
mkdir build
cd build
cmake ..
make
```
```bash
./my_project
```
